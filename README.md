# SPC support for Roaming Authenticators

[Secure Payment Confirmation](https://github.com/w3c/secure-payment-confirmation) (SPC) is a Web API to support streamlined
authentication during a payment transaction being developed by [W3C Web Payments Working
Group](https://www.w3.org/Payments/WG/). [Roaming authenticators](https://www.w3.org/TR/webauthn-2/#roaming-authenticators)
is a class of authenticators that are removable from, and can "roam" between, client devices. Support for roaming authenticators is part of the [SPC requirements](https://github.com/w3c/secure-payment-confirmation/blob/main/requirements.md#fido-considerations)
