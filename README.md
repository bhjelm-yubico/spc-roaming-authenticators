# SPC Support for Roaming Authenticators
## Overview
Secure Payment Confirmation (SPC) (https://github.com/w3c/secure-payment-confirmation/blob/main/README.md) is a Web API to support streamlined authentication during a payment transaction. Roaming authenticators (https://www.w3.org/TR/webauthn-2/#roaming-authenticators) is a class of authenticators that are removable from, and can "roam" between, client devices. SPC support for roaming authenticators is part of the requirements (https://github.com/w3c/secure-payment-confirmation/blob/main/requirements.md#fido-considerations).
