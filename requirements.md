# SPC Support for Roaming Authenticators - Requirements and Design Considerations
## Overview
Secure Payment Confirmation (SPC) (https://github.com/w3c/secure-payment-confirmation/blob/main/README.md) is a Web API to support streamlined authentication during a payment transaction. Roaming authenticators (https://www.w3.org/TR/webauthn-2/#roaming-authenticators) is a class of authenticators that are removable from, and can "roam" between, client devices. Support for roaming authenticators is part of the SPC requirements (https://github.com/w3c/secure-payment-confirmation/blob/main/requirements.md#fido-considerations).

## Terminology
The use of terms "MUST", "SHOULD", and "MAY" are defined in [RFC2119] (https://datatracker.ietf.org/doc/html/rfc2119).
